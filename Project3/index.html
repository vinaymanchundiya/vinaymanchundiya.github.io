CS 491 - Project 3

Introduction

How we came up with the idea ?

For 3rd project of CS-491, we decided to work on an Augmented Reality game which is engaging and is based on the collaborative experience design principles. While deciding for the project, we wanted to create something that is easy to use and something that non tech savvy people can also use. Keeping that in mind, we decided to create battle royale themed tank battle. This game is inspired by the combination of game which is inspired by a legacy game titled Battle City and the theme of battle royale of popular mobile game PlayersUnkown's Battleground. Owing to its design, this game provides the nostalgic experience of playing a game which we grew up playing in a modern game design.

What is this game about ?

As mentioned earlier, this game is inspired from Battle City, each player will be assigned a tank. The motive of the game is to kill opponent tanks. The last man standing will be deemed the winner. The game will spawn many power ups which when redeemed by a player will provide some advantages to the player. Also, there are different kinds of obstacles in the game which will have impact on the tactical approach to the game.

Game features

Following are the key features of the game:

  • Can only be played on iOS devices
  • Uses the Augmented reality technology for the user to play it
  • Based on multipeer connectivity
  • 2-4 players can play this game
  • Features 4 levels to choose from

How to install the game

To install this game download the game source code from Add drive link here. Unzip the file and open the project in xCode. Maek sure you have Apple developer account. If you do not have the account then you will need to create one. After loading the code into the system, attach your iOS device where you want to install the game. Click on run button and the game will be installed on your iOS device. Reeat the process for other devices as well.
