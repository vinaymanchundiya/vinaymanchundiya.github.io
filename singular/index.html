Project Description

So, here we are with another VR project, we call it- Hotel California...... You can find the download links to the project files and the executable file after this description. We have also included the instructions to load the executable file and enjoy the whole game. The game demonstrates a halloween theme in which you can encounter different ghost characters in each room of Hotel California. The first room is the Base room which is basically entrance of the hotel, where you will encounter napster, who is casper's cousin from another world. Napster is gonna show some nice tricks. The second room is the Caregiver's room where you have to find sphere and complete the pentagram and walk into the closet guarded by the CareGiver and close the door. The door then opens into the second room which belongs to Gabriel vampire. Gabriel stands very angrily in the room as his girlfriend's portrait is on the floor. On placing the portrait properly, Gabriel flys away and gives in a cave in the room which opens into another room in which a zombie resides. The zombie is supposed to be trapped in the bear trap using the meat loaf lying on the floor. Once Zombie is trapped a handle becomes visible which can be used to pull the player on the top open floor of the hotel where stands an IceGolem. The cage of the golem hangs straight above him which is tied by a rope. The rope when set on fire makes the cage fall on the golem which ends the game.

Learning and Insights

This was indeed a fun project. We spent a lot of time on creating an immersive graphic experience which is scary and fun at the same time. Bound by the restirctions of the available space, we had to be very creative about choosing the design of the mini-world that we tried to create. From navigation point of view, we had to created portals,tunnels and lifts to transport the user to new location. We chose a pretty simple theme of a scary hotel which gave us creative freedom to choose the inhabitants of the room. With respect to introduction for the user to the world, we created a base room which was almost empty but gave the user look and feel of the world they will be going to be in. Then we took the user to scary looking rooms where they had to solve a mini-puzzle to go the next level. The challenges included helping a wizard to perform a ritual to caging a beast by burning a rope. We also included fun interaction for the user to play with.

Since, the duration of the project was small(1 month), there is a lot of room for improvement. Given that we have enough amount of time and resources, we could have done things a lot differently and made this small project into a complete AAA level VR game with high definition graphics.
